# What is this?

```js live=true
const r = $rect(50, 50);
const c = $circle(50);

$animate(
  { item: r, width: 100, duration: 1 },
  { item: c, width: 100, x: 100, y: 100, duration: 1, fill: "blue" }
);

$animate({ item: c, y: 0, fill: "red" });

$text("Nice!", 100, 100, { fontSize: 24 });
```

This is a work in progress. Some day it will be a cool site about learning JavaScript using an interactive drawing
 setup.
 
 Static code:
 
 ```js
const f = (x) => 2 * x;

$text(f(13));
```

It will be based around code-samples like the following:

```js
$rect($stageWidth/2, $stageHeight/2, $stageWidth/4, $stageHeight/4, {
  fill: "purple"
});

$text("Hello world!", $stageWidth/2 - 45, $stageHeight/2, {
  align: "center",
  fill: "white"
});

$log("That's neat!")
```
