# What is this?

```js live=true
// Rectangle
const r = $rect(
  50,
  50,
  $stageWidth/2,
  $stageHeight/2,
  { offsetX: 25, offsetY: 25 }
);

// And text
const t = $text(
  "BAM!",
  $stageWidth / 2,
  $stageHeight / 2 - 25,
  { align: "center", offsetX: 25, offsetY: 25, fontSize: 0 }
);

// Start the spin
$animate({ item: r, rotation: -30, duration: 1, cornerRadius: 10 });

// Then spin more and swing text in.
$animate(
  { item: r, rotation: 360, duration: 1, cornerRadius: 0, fill: "green" },
  { item: t, fontSize: 24, duration: 0.5 }
);
```

This is a work in progress. Some day it will be a cool site about learning JavaScript using an interactive drawing
 setup.
 
 Static code:
 
 ```js
const f = (x) => 2 * x;

$text(f(13));
```

It will be based around code-samples like the following:

```js
$rect($stageWidth/2, $stageHeight/2, $stageWidth/4, $stageHeight/4, {
  fill: "purple"
});

$text("Hello world!", $stageWidth/2 - 45, $stageHeight/2, {
  align: "center",
  fill: "white"
});

$log("That's neat!")
```
